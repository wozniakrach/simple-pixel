CREATE!

Make your own pixel art using the create page! Clicking or moving over a pixel while holding the mouse down will color the pixel. Use the palette on the left to change colors and the buttons at the bottom for other functions.

  • Download: This button will download your current canvas.
  • Add Color: Select this to choose a new color to add to the palette.
  • Submit: Submit your current canvas to (potentially) be added to the gallery! You can enter an optional title and/or artist name. (By using this button you give permission for any art on the current canvas to be displayed on this site.)

To view more art created with this site, check out the gallery. To submit feedback or learn more, see the about page.
